# Anti-Delivery Technique: Detect & Block Phishing

## Purpose
Detects and blocks malicious files delivered via phishing emails or fake websites.

## Workflow
1. Scans incoming downloads and email attachments.
2. Identifies known spyware signatures or suspicious patterns.
3. Blocks or quarantines detected threats.

## Key Files & Functions
- `function1_detect_spyware.py`: Scans for spyware files.
- `function2_scan_spyware_files.py`: Deep scan for suspicious files.
- `function3_quarantine_spyware.py`: Quarantines detected threats.

## Usage
Run these scripts to scan and protect the system from phishing-based delivery attacks.
