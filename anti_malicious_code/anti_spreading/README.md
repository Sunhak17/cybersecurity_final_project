# Anti-Spreading Techniques

## Purpose
Monitors for suspicious network activity and blocks malware spreading or data exfiltration attempts.

## Workflow
1. Monitors outgoing and incoming network connections.
2. Detects abnormal or unauthorized activity.
3. Blocks or reports spreading/exfiltration attempts.

## Key Files & Functions
- `function7_monitor_exfiltration.py`: Monitors for data exfiltration.
- `function8_block_spreading.py`: Blocks spreading attempts.
- `function9_generate_report.py`: Generates incident reports.

## Usage
Run these scripts to monitor, block, and report on spreading and exfiltration activities.
